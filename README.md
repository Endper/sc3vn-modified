# SC3's Visual Novel Engine

This library is designed to make it easy for beginners to make a Visual Novel

## Installation

Use the package manager [pip](https://pip.pypa.io/en/stable/) to install SC3's Visual Novel Engine.

```bash
pip install sc3vn
```

## Usage

```python
import sc3vn
```
